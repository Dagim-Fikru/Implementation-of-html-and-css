Work flow of projects

1/ Plan

The first step in work flow is planning, Here i create the data that allows a customer to judge how the entire site will look like.

2/ Design

The second step will be designing, Here i will give a shape to the website. All the visual content, such as images, photos, and videos is created at this step. After this i will go to excuting.

3/ Coding

The third step is excuting what i designed so far, Here i will start creating the website it self. Graphic elements that have been designed during the previous stages should be used to create an actual website. After doing this i will go to the next step.

4/ Test

The last step is Testing what i have done in this step Every single link should be tested to make sure that there are no broken ones among them. I will check every form, every script, run a spell-checking software to find possible errors.If any error is observed i directly go to the previous steps.
